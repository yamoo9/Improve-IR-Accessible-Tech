IMG Element 사용

접근성 O | 성능 X

<img> 요소 사용시에는 Windows 고대비 모드(단축키: Alt + Shift + PrintSc)에서도 화면에 표시됩니다. 접근성에는 문제가 없지만, 전체적인 성능에서는 문제가 됩니다.

CSS Background Image 사용

접근성 X | 성능 O

CSS 배경이미지 기법(Phark Method & WA IR)을 사용한 경우, Windows 고대비 모드에서는 화면에 배경 이미지가 표시되지 않아 접근성 준수에 문제가 됩니다.

CSS Background Image & improveIR.js 사용

접근성 O | 성능 O

Windows 고대비 모드에서 CSS 배경이미지 기법이 가지는 접근성 준수 문제를 해결하기 위해서 스크립트(improveIR.js)를 사용하여 Windows 고대비 모드인지를 체크하여 화면에 배경이미지 대신 텍스트를 보여줘 접근성을 준수할 수 있습니다.
